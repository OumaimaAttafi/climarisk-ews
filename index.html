CLIMARISK-EWS

Machine Learning Early Warning System for Climate-Induced Financial Risk

Project Overview

This project develops a machine learning-based Early Warning System (EWS) to predict financial distress in the banking sector using climate risk indicators and financial variables.

Objective

To estimate the probability of banking instability by integrating environmental and financial data, enabling proactive risk management and policy insights.

Key Features

  • Climate risk indicators (temperature, drought, emissions)
  • Financial performance metrics (ROA, ROE, Z-score)
  • Machine learning models (Logistic, Random Forest, XGBoost)
  • Explainable AI (SHAP analysis)
  • Early warning probability output

How It Works

The model takes climate and financial inputs and outputs the probability of financial distress. It helps identify vulnerable systems before crises occur.

Repository Structure

/data – Dataset
/notebooks – Google Colab notebooks
/src – Model scripts
/app – (Optional) Web dashboard
